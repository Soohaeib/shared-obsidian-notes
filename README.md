# Public study site

This repository contains a static copy of the site content for GitHub Pages hosting.

## Publish as a website

1. Create a new GitHub repository.
2. Push this repository to GitHub.
3. In GitHub, open Settings → Pages.
4. Choose GitHub Actions as the deployment source.
5. The included workflow will publish the site automatically.

The site entry point is the root index page, which redirects to the main content under `4th-semester/`.
